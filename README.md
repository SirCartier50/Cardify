# Cardify — AI Flashcard SaaS

An AI-powered flashcard generator with a public deck marketplace and study analytics. Built with Next.js, React, Clerk, Firestore, OpenRouter API, and Stripe.

## Features

- **AI Flashcard Generation** — Paste any text and get study-ready flashcards generated by Llama 3.1 via OpenRouter
- **Public Deck Marketplace** — Share your decks with the community, browse and clone decks created by others
- **Study Mode** — Flip through cards one at a time, mark as correct/missed, and track your accuracy
- **Analytics Dashboard** — View study streaks, mastery rates per deck, session history, and total cards studied
- **Stripe Subscription Billing** — Free tier (10 cards/generation) and Premium tier ($3.99/mo, 40 cards/generation)
- **Clerk Authentication** — Secure sign-in/sign-up with Clerk

## Tech Stack

- **Frontend:** Next.js 14, React, Material-UI
- **Backend:** Next.js API Routes
- **Database:** Firebase Firestore
- **Auth:** Clerk
- **AI:** OpenRouter API (Llama 3.1)
- **Payments:** Stripe
- **Deployment:** Vercel

## Getting Started

### Prerequisites

- Node.js 18+
- A [Clerk](https://clerk.com) account
- A [Firebase](https://firebase.google.com) project with Firestore enabled
- An [OpenRouter](https://openrouter.ai) API key
- A [Stripe](https://stripe.com) account

### Setup

1. Clone the repo:
   ```bash
   git clone <repo-url>
   cd Cardify
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Copy `.env.example` to `.env.local` and fill in your keys:
   ```bash
   cp .env.example .env.local
   ```

4. Update `firebase.js` with your Firebase project config.

5. Run the dev server:
   ```bash
   npm run dev
   ```

6. Open [http://localhost:3000](http://localhost:3000)

### Deploy to Vercel

1. Push to GitHub
2. Import the repo on [Vercel](https://vercel.com)
3. Add all environment variables from `.env.example` to Vercel's project settings
4. Deploy
